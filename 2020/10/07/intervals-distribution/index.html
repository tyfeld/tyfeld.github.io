语言统计分析：长数字序列相邻相同数字间的距离分布

1. 简述

《语言统计分析》的第一次作业：统计在一个长的无规律的数字序列（无理数的小数点后若干位，如 π,e,2,ln⁡2\pi,e,\sqrt {2}, \ln {2}π,e,2​,ln2 等等无理数小数点后 10^(5) 位以上）中，相邻相同数字间的距离分布。

2. 得到数据

首先使用 python 来获取距离数据。
读入数字序列文件并且转换成列表来便于统计：

## 读入文件，并转换成其中所有数字的列表 
with open ("1bypi.txt",'r') as f:
    content = f.read ()
lines = content.split ("\n")
digits = []
for line in lines:
    digits += line
## 获得一个间距序列 intervals_of_0
position_of_0 = -1
intervals_of_0 = []
for index in range (len (digits)):
    if digits [index] == '0':
        if position_of_0 != -1:
            intervals_of_0.append (index - position_of_0)
        position_of_0 = index

这样在数组 position_0f_0 中就储存了所有的间隔距离。

3. 绘图

使用同样的方法对类似的数字进行统计：选取 1/pi 和自然对数 e 的小数点后 100000 位，统计前者数字 0,1,2 距离分布和后者数字 5 的距离分布，绘制频率（概率）分布图：

import seaborn as sns

从如上频率（概率）分布图中，可以明显观察到：

  • 相邻数字的距离分布总体上呈现明显的下降趋势。相邻数字间距为 1 的概率最大，大约在 0.10 左右，随后间距增大，概率减小。
  • 总体的减小趋势呈现出指数减小的特征。

4. 规律拟合

尝试将概率密度曲线拟合成函数，大约有以下两种方法：

方法一：曲线回归

由于原曲线呈现明显的指数函数形态，将 y 轴取对数后观察图像：
在原代码中添加

plt.yscale ('log')

得到的第一张图图像为

可以看到图像明显回归到直线。尝试使用回归分析就能得到结果。
当做到这一步时，想到了这个问题中的条件概率，发现可以轻松通过数学分析推导出这个指数函数。

方法二：数学分析

假定给定了一个数字 0，目前在序列的一个特定位置。
那么相邻的下一个 0 的间距为为 xxx 的概率，实质上就是在上述前提条件下，中间 (x−1)(x-1)(x−1) 个数都不为 0，第 xxx 个数为 0 的概率，如果把所有数字都看作等概率随机生产，那么这个概率即为 0.9x−1∗0.10.9^{x-1}*0.10.9x−1∗0.1。
而这个概率也就是在所有的相邻的数字 ‘0’ 的间隔中，间隔为 xxx 的概率。
因此对于间距 xxx，预测的概率函数为 $$P (X=x)=0.9^{x-1}*0.1$$
这个函数已经满足了发现的两条曲线规律，只需要在原图中加上这一条预测的函数曲线

x = np.arange (0,140,1)
y = 0.1*(0.9**(x-1))
plt.plot (x,y,color='green',label='Predicted Function')

得到图像：

图像几乎完美地拟合！
也因此证明了我们的数学推导是正确的。

5. 总结

通过统计分析和绘图，我们得到了这样的结论：对于一个非常长的无规律数字序列（看作每一个数都是等概率随机产生），相邻的相同数字的距离分布概率曲线渐进分布于函数曲线 P(X=x)=0.9x−1∗0.1P (X=x)=0.9^{x-1}*0.1P(X=x)=0.9x−1∗0.1，其中 xxx 代表间隔的距离，概率 PPP 为在所有的间隔中，值为 xxx 的间隔出现的概率。
